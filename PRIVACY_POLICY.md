# platalea 隐私政策

最近更新日期：2026年3月31日

欢迎使用 platalea。

platalea 是一款以本地文件管理、加解密、压缩解压、文档查看、局域网文件传输、音乐播放和基于 Git 的数据同步为主要功能的 Android 应用。我们重视你的个人信息和数据安全。本隐私政策用于说明本应用在你使用过程中可能涉及的信息处理方式。

## 一、我们处理的信息范围

platalea 主要定位为本地工具软件。除你主动发起的功能操作外，本应用不会主动收集与功能无关的个人信息。

根据你使用的具体功能，本应用可能处理以下信息：

1. 你主动选择的文件、目录和文档树访问权限。
2. 你主动导入、创建、编辑、加密、解密、压缩、解压、播放或查看的本地文件内容。
3. 你主动配置的应用参数，例如根目录、最近打开记录、界面偏好、播放列表、词典数据、缓存信息、Git 仓库地址、FTP 服务配置等。
4. 你主动输入的密码、密钥口令、Git 认证信息或 FTP 访问密码。
5. 为实现局域网传输、Git 同步、文件预览、缓存和后台播放所必需的临时运行数据、日志信息和缓存文件。

## 二、我们如何使用这些信息

本应用处理相关信息仅用于实现你主动使用的功能，包括但不限于：

1. 访问并管理你授权范围内的本地文件和目录。
2. 执行文件浏览、复制、移动、删除、压缩、解压、查看和编辑操作。
3. 执行 GPG 加密解密、ZIP 加密、快速混淆等安全处理功能。
4. 在局域网环境下提供 FTP 文件传输服务。
5. 在你主动配置后访问你指定的 Git 仓库，以便下载、同步或提交共享数据。
6. 播放你选择的本地音频文件，并在系统通知中显示播放控制状态。
7. 在应用内部保存必要配置，以便下次继续使用相关功能。
8. 生成缓存文件，以提升压缩包浏览、文档渲染、电子书查看和词典查询等功能的性能。

## 三、权限使用说明

本应用当前申请的 Android 权限主要包括：

1. 网络权限：用于 FTP 服务、Git 同步、访问局域网或互联网资源。
2. Wi-Fi 状态权限：用于辅助网络传输相关功能判断当前网络环境。
3. 前台服务权限：用于音乐播放等需要在后台持续运行的功能。
4. 媒体播放前台服务权限：用于保持音频播放服务稳定运行。
5. 通知权限：用于显示播放控制、前台服务状态等系统通知。

说明：

1. 文件访问主要基于 Android 系统提供的文档树授权机制，由你主动选择授权目录。
2. 本应用不会在未经你主动授权的情况下扫描你的整个设备文件。
3. 某些网络能力仅在你主动开启 FTP 服务或主动配置 Git 同步时才会使用。

## 四、数据存储方式

你的数据主要存储在以下位置：

1. 你自行选择并授权的本地目录。
2. 应用私有沙箱目录中的配置、缓存、密钥材料、索引数据、临时文件和运行数据。
3. 你主动配置的外部目标位置，例如局域网 FTP 客户端连接方或你指定的 Git 仓库。

本应用不会默认将你的文件内容上传到开发者自有服务器。

## 五、数据共享与对外提供

除以下情形外，我们不会主动向第三方提供你的信息：

1. 你主动启用 FTP 服务并在局域网中向其他设备提供文件访问。
2. 你主动配置 Git 仓库地址并执行拉取、提交、推送等同步操作。
3. 你主动选择使用其他应用打开文件，或通过系统分享能力将文件交由其他应用处理。
4. 法律法规、司法机关或行政主管部门依法要求提供。

请注意，当你主动使用 Git、FTP 或“用其他应用打开”等功能时，相关数据处理将受到目标平台、目标设备或第三方应用自身规则的约束。

## 六、关于加密、密码和敏感信息

本应用提供加密相关功能，但你仍需理解以下事项：

1. 你输入的密码、密钥口令和敏感文本仅用于你主动发起的加解密、解压或认证操作。
2. 私钥、配置和部分安全相关数据可能保存在应用私有目录中，以支持本地使用。
3. 即使应用已采取本地保护措施，移动设备运行环境、恶意软件、Root、截屏、剪贴板、输入法、系统漏洞等因素仍可能带来风险。
4. 你应妥善保管密码、密钥和重要数据备份；一旦遗失，相关数据可能无法恢复。

## 七、缓存与日志

为提升性能和稳定性，本应用可能在设备本地生成缓存文件、导入中间文件、文档渲染缓存、电子书缓存、压缩包解压缓存及必要的错误日志。

其中：

1. 非加密内容的部分缓存可能会保留一段时间，用于提升重复打开速度。
2. 某些加密内容在查看结束后会自动清理缓存。
3. 错误日志和提示信息主要用于本地故障定位，不用于广告画像或商业分析。

## 八、未成年人保护

本应用主要面向具备文件管理和数据处理能力的用户。若你是未成年人，请在监护人指导下使用本应用，并由监护人阅读和确认本隐私政策。

## 九、第三方组件说明

本应用可能集成开源第三方组件以实现加密、压缩、文档解析、Git 同步、FTP 服务等技术能力。这些组件用于实现对应功能。本应用当前不以内置广告 SDK、商业统计 SDK 或个性化推荐 SDK 为主要功能基础。

但当你主动连接外部 Git 服务、使用系统浏览器、调用其他应用打开文件时，相关行为可能受到对应第三方服务提供方隐私政策约束。

## 十、你的权利

你可以通过以下方式管理你的数据：

1. 撤销应用的目录访问授权或系统权限授权。
2. 删除应用内配置、缓存、本地导入数据或卸载应用。
3. 停止使用 Git、FTP、播放通知等相关功能。
4. 自行删除你保存在本地目录、应用私有目录或外部同步目标中的数据。

请注意，删除或卸载后，部分未备份的数据可能无法恢复。

## 十一、隐私政策的更新

如果本应用功能、权限使用范围或数据处理方式发生实质变化，我们可能更新本隐私政策。更新后的内容将以应用内展示、发布页说明或其他合理方式向你说明。
