# 调试说明

## 使用仿真环境

（需要下载android自己的sdkmanager，不能用debian的那个）
sdkmanager "system-images;android-36;default;arm64-v8a"
avdmanager create avd -n lmdev -k "system-images;android-36;default;arm64-v8a" -d pixel_6

emulator -avd lmdev
adb install -r app/build/outputs/apk/debug/app-debug.apk
或者
gradle installDebug
