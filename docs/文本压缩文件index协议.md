# 文本压缩文件index协议

.llm.zip文件默认用epub阅读器打开，里面每个文件看作是书的一个章节，但顺序是不可指定的。

本协议提供一种说明章节的方法：如果.llm.zip的压缩包中，在第一层目录下包含.index文件，那么该文件看作是.llm.zip的章节定义文件。

.index文件的格式定义如下：

* .index为utf-8格式的文本文件，按顺序每行定义一个章节
* 每行包含两个部分：章节文件路径，章节文件标题，中间用空格隔开。如果章节文件中有空格，可以用单引号或者双引号把整个目录引用起来，这种情况，整个引号内的内容都看作是路径的一部分，路径之后剩余的部分，都看作章节文件标题，即使有空格，也看作是标题
* 如果章节文件标题部分没有内容（或者只有白字符），那么章节文件路径就看作是章节标题
