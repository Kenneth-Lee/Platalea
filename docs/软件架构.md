# 软件架构

程序实现在com.kenny.localmanager中，主Activity按一般习惯，是默认的MainActivity。
绘制先按管理先绘制LocalManagerTheme，用来设定配色等基本设置，然后在这个Theme上
绘制真正的界面，FileBrowserApp()->FileBrowserAppScreen()。

## 主界面

FileBrowserAppScreen()是融合界面，定义一组

目录内「只选不开」的临时拣选行为见 `docs/目录拣选模式.md`（`DirectoryPickRequest`、
拣选模式 Tab 切换、独立 `pickList`）。
